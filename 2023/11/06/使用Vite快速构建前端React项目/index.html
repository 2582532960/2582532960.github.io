五、样式

reset.css 是一种 CSS 文件，它的作用是重置网页的默认样式。由于不同浏览器对于 HTML 默认样式的设定不同，导致网页在不同浏览器上的显示效果各异。为了避免这种情况，可以使用 reset.css 对网页默认样式进行重置，使不同浏览器上的显示效果尽量一致。
